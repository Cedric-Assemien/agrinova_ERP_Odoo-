# Images AGRINOVA

## Instructions

Pour finaliser le module, vous devez ajouter manuellement les images suivantes:

### 1. Icon du module
**Chemin**: `static/description/icon.png`
- **Dimensions**: 256x256 pixels
- **Format**: PNG avec transparence
- **Contenu**: Logo AGRINOVA ou icône représentant l'agroalimentaire
- **Suggestions**: Fruit, plant, leaf ou logo de l'entreprise

### 2. Bannière du module
**Chemin**: `static/description/banner.png`  
- **Dimensions**: 560x280 pixels
- **Format**: PNG
- **Contenu**: Bannière promotionnelle du module
- **Texte suggéré**: "AGRINOVA - Gestion Intégrée Agroalimentaire"

## Génération des images

Vous pouvez:
1. Utiliser le logo existant d'AGRINOVA
2. Créer des icônes sur https://www.canva.com
3. Utiliser des icônes gratuites sur https://www.flaticon.com
4. Demander à un designer

## Couleurs recommandées

- **Vert**: #2e7d32 (agriculture, nature)
- **Orange**: #ff9800 (transformation, énergie)
- **Bleu**: #2196f3 (qualité, confiance)

## Note

Le module fonctionnera sans ces images, mais elles amélioreront l'apparence dans la liste des applications Odoo.
